# Java Coffee v1.0

#### Java Study Note  
----------- 
![JavaLogo](http://www.javaguru.co/wp-content/uploads/2015/02/java.jpg)
	
	이 문서는 필자의 Note이자, Java 문법을 쉽고 자세하게 알려주는 것을 목적으로 한 비영리 
	기본서입니다.
	
	Opentutorials.org (생활코딩)의 목표인 
	
	프로그래머들이 이룩한 성취를 일반에 알려서 고독한 프로그래머들의 관객을 모객하는 활동에
	
	감명을 받아 제작합니다. 
	필자 또한 Java의 세상에서 한걸음씩 나아가고 있는 사람의 관점으로 기록합니다.
	
	이 문서의 목표는 Coffee를 마시듯 초심자에게 부담없이, 하지만 coffee의 향을 진하게 
	느끼실 수 있도록 상세한 설명을 기록 하였습니다. 
	
	그리고 생활코딩 Egoing님의 Java 강의 + 윤성우님의 '난 정말 Java를 공부한 적이 없다구요' 강의의 
	장점들만을 모아 순도 90%의 Java Coffee를 만들었습니다.
	
	초심자에게는 초심자에게 맞는 이해와 설명이 필요합니다. 
	시간이 지나고 경험치가 쌓이면 그 설명들은 레고블럭을 조립하듯이 다시 조립되고, 
	새로이 이해하게 됩니다. 이 관점에서 이 문서도 지속적으로 Update 계획이 있습니다.
